WebViewCustomProtocol
=====================

Sample iOS project showing two-way communication between Objective-C
and Javascript in a UIWebView. "Protocol" here refers not to an obj-c
protocol, but to a custom URL protocol.




---------------------------------------------------------------------
This project was developed as sample code for the iOS Developer Study
Group that meets at the Hacker Dojo in Mountain View, CA.
